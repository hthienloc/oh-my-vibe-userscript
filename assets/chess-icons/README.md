# Chess.com Icons Assets

Please upload the official evaluation icons here. The userscript is configured to look for these specific filenames:

| Classification | Filename |
| :--- | :--- |
| Brilliant | `brilliant.png` |
| Great Move | `great.png` |
| Best Move | `best.png` |
| Good Move | `good.png` |
| Book Move | `book.png` |
| Inaccuracy | `inaccuracy.png` |
| Mistake | `mistake.png` |
| Missed Win | `miss.png` |
| Blunder | `blunder.png` |

**Format**: Recommended PNG with transparent background, 40x40 or 64x64 pixels.
